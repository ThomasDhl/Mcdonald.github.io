# 🍔 扫码领免费麦当劳？

> 你这个乐乐，真以为扫二维码能有免费麦当劳吃？

## 项目说明

这是一个恶搞二维码项目，由两个页面组成：

### 📱 页面结构

| 页面 | 文件 | 说明 |
|------|------|------|
| 助力页 | `help.html` | 伪装成"好友助力领汉堡"活动，诱使对方扫码 |
| 揭晓页 | `index.html` | 恶搞页面，扫码后显示真相 |

### 🔗 套路流程

1. 把 `help.html` 页面展示给目标
2. 对方看到"扫码助力好友即可免费领取汉堡"
3. 扫码后跳转到 `index.html`
4. 收到意料之外的惊喜 🎉

### 📂 文件说明

```
QR/
├── help.html              # 助力页
├── index.html             # 恶搞揭晓页
├── qrcode_with_logo.png   # 带麦当劳Logo的二维码
├── lele.jpg               # 目标照片
├── logo.png               # Logo素材
├── gen_qr.py              # 二维码生成脚本
└── README.md
```

### 🛠️ 修改二维码

如需重新生成二维码，修改 `gen_qr.py` 中的 URL 后运行：

```bash
python gen_qr.py
```
